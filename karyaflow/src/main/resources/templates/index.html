KaryaFlow is a team and project management backend API. Use this API to create, update, manage, and delete projects, assign team members, and track progress. This documentation provides a quick overview of available endpoints and their usage.

Endpoints

GET /api/project/get-all-projects
Fetch a list of all available projects.
Status Codes: 200 OK, 204 No Content
POST /api/project/create-project
Create a new project with title, description, start date, and team members.
Body Parameters (JSON):
  • title: string
  • description: string
  • startDate: string (format: YYYY-MM-DD)
  • teammembers: array of strings
Status Codes: 201 Created, 400 Bad Request
GET /api/project/get-specific-project/{id}
Fetch a specific project using its ID.
Path Parameter: id (Long)
Status Codes: 200 OK, 404 Not Found
PUT /api/project/update-project/{id}

Path Parameter: id (Long)
Body Parameters (JSON): same as POST
Status Codes: 200 OK, 400 Bad Request, 404 Not Found
DELETE /api/project/delete-projects/{id}
Delete a specific project by its ID.
Path Parameter: id (Long)
Status Codes: 204 No Content, 404 Not Found
